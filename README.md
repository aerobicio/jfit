# Jfit

A JRuby library for parsing FIT files. Uses the http://thisisant.com FIT SDK.

## Installation

Add this line to your application's Gemfile:

    gem 'jfit'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jfit
